# Contributing to MAGO

We welcome contributions! Please follow our code of conduct and submit pull requests against the main branch.

## Development Setup

1. Install Go 1.20+ and Python 3.10+.
2. Clone the repo.
3. Run `make build` and `make test` to verify everything works.
4. For Python generator, install dependencies with `pip install -r tools/rulesgen/requirements.txt`.

## Code of Conduct

Be respectful and professional. Harassment or abusive behavior will not be tolerated.
